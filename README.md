🔤 Ahorcado — Juego en JavaScript
Juego clásico del Ahorcado, desarrollado con HTML, CSS y JavaScript.
Incluye teclado visual, control de errores y normalización de tildes.

✨ Funcionalidades
Selección aleatoria de citas de alexelcapo (probablemente esto cambie en un futuro, pero hace demasiada gracia).

Normalización de tildes para evitar errores al introducir letras.

Teclado visual interactivo.

Contador de fallos y detección de victoria/derrota.

Interfaz simple y responsive.

🚀 Cómo jugar
Clona el repositorio:

Código
git clone https://github.com/mothalbete/ahorcado
Abre index.html en tu navegador.

Haz clic en las letras del teclado o usa tu teclado físico.

Adivina la cita antes de que se te agoten los intentos
